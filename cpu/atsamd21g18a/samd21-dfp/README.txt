This directory contains a partial copy of the device support pack from Microchip/Atmel for the
SAMD21 microcontroller series, which is used to add HAL support for their devices to this project.

The files are kept in their original form and therefore retain their original Apache license. We
avoid modifying them so updating becomes a simple task of copying the new files over the older
ones.

Please observe the original license of the files when using this project. The complete pack was
downloaded from here:

http://packs.download.atmel.com/

The files contained here correspond to version 1.2.280 of the SAMD21 series device support package.
